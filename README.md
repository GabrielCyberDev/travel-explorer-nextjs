# ✈ Travel Explorer

Projeto desenvolvido com **Next.js + TypeScript**, simulando um portal de viagens com listagem de destinos e páginas dinâmicas.

---

## 🚀 Demonstração

Página inicial com destaque para destinos turísticos e navegação dinâmica entre páginas.

---

## 📸 Preview

- Home com hero image
- Listagem de destinos
- Página dinâmica de detalhes

---

## 🧰 Tecnologias utilizadas

- Next.js
- React
- TypeScript
- CSS Modules
- next/link
- next/image

---

## 📁 Estrutura do projeto
src/
├── components/
├── data/
├── pages/
├── styles/
public/
├── images/


---

## 🌍 Funcionalidades

- Listagem de destinos turísticos
- Página dinâmica para cada destino
- Layout reutilizável
- Componentização
- Estilização com CSS Modules
- Responsividade
